Proyectos realizados

Diseño de sala de conciertos

El proyecto realizado consiste en el diseño de una sala de conciertos para eventos sin amplificación de sonido, con capacidad para 1169 personas.

Circuito de distorsión para guitarra eléctrica

En este proyecto se desarrolló un circuito para guitarra eléctrica que logra generar distorsión mediante la saturación de transistores BJT.

Estudio de grabación y mezcla

Se realizó el diseño de un estudio de grabación y mezcla de índole profesional, completamente equipado.

Circuito compresor de audio

Se llevó a cabo un análisis del funcionamiento de un circuito compresor de audio, al cual se le realizaron modificaciones y luego se diseñó su placa PCB.

Voltímetro activo para baja señal

El proyecto consiste en el diseño y fabricación de un voltímetro activo para señales de baja amplitud.
